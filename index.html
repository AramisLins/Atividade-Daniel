HTML (Linguagem de Marcação e Hipertexto) é o bloco de construção mais básico da web.Define o significado e aestrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação(CSS) ou a funcionalidade/comportamento(JavaScript) de uma página da WEB.

Curiosidades:

O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

HTML é o elemento raiz do documento HTML.
HEAD contém informações do documento, como o título da página e referências a arquivos CSS e JavaScript.
TITLE define o título da página que é exibido na guia do navegador.
BODY é o contêiner para o conteúdo da página.
HEADER contém informações de cabeçalho para a página, como o título e o menu de navegação.
NAV define um conjunto de links de navegação para a página.
UL define uma lista não ordenada de itens.
LI define um item em uma lista.

O que é CSS ?

Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags . Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos.

O que é REACT?

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

O que é JavaScript?

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
